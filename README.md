# pokedex
A Pokedex app using Pokeapi and Alamofire. (iOS10, Swift3)

Including: 
- Custom fonts, playing audio with AVAudioPlayer
- Downloading and parsing JSON data with Alamofire from Pokeapi
- Parsing csv
- UICollectionsViews
